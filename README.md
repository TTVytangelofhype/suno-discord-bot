# Suno Discord Bot

A Discord music bot designed for playing Suno music links in a Discord voice channel.

This bot was created by **TTVytangelofhype** and is built for Discord servers that want a simple Suno-focused music bot with queue support, playlist commands, and server-friendly setup.

---

## Features

- Play Suno music links in Discord voice channels
- Queue system
- Skip, stop, leave, and now playing commands
- Playlist save/load/delete support
- DJ role support
- Guild/server lock support
- Optional upload and direct MP3 blocking
- Lavalink music playback support
- Windows VPS and Linux VPS friendly
- PM2 compatible for 24/7 hosting

---

## Requirements

Before using the bot, you need:

- Node.js 18 or newer
- npm
- A Discord bot token
- A Discord server
- Lavalink server/node
- FFmpeg installed if required by your setup
